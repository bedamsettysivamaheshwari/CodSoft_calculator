# Simple Calculator

## Description
This is a simple calculator application developed in Python. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division of two numbers
- Handles division by zero errors
- User-friendly command-line interface

## Technologies Used
- Python 3

## How to Run
1. Install Python on your system.
2. Download or clone this project.
3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the following command:

```bash
python calculator.py
```

6. Enter two numbers and choose an operation (+, -, *, /).
7. View the result displayed on the screen.

## Sample Output

```
Enter first number: 10
Enter second number: 5
Choose operation (+, -, *, /): *

Result: 50
```
